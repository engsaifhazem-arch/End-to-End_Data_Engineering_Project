# Business Analysis

## Project Overview

This document outlines the business analysis phase of the End-to-End Data Engineering Project.

## Key Deliverables

- Product Requirements Document (PRD)
- Data Requirements Document (DRD)

## Project Phases

1. **Business Analysis** (Current Phase)
2. Landing Layer & Staging Layer
3. Star Schema Data Warehouse
4. ETL Pipelines
5. Semantic Layer
6. Interactive Dashboard with KPIs

## Data Source

- Olist E-commerce Dataset

## Technology Stack

- Microsoft SQL Server
- SQL Server Integration Services (SSIS)
- SQL Server Analysis Services (SSAS)
- Power BI

The project delivers 10 key business KPIs through an interactive dashboard powered by the Microsoft BI Stack.
